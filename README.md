# RollinDD Platform Starter

RollinDD is a multi-tenant playlist-site launcher. One GitHub repo and one Vercel project can power many public music websites, each driven by a unique Suno playlist, custom theme, lyric search index, and optional domain.

See `docs/TECHNICAL_OVERVIEW.md`, `docs/CENTRAL_COMMAND_SPEC.md`, and `docs/DEPLOYMENT_STEPS.md`.

Important: this public repository must never contain real API keys, tokens, private Suno account data, or credentials. Put secrets only in Vercel Environment Variables.
